# Rock Paper Scissors Game with AI
This project is a simple rock-paper-scissors game with using pygame. 
The player can play against to AI.
Whole project is written with python and it has simple graphical interface. 

# Features
Simple pygame interface
Rule-based AI that analyzes player's history.
Visual display of win, lose, and draw states.
Dynamic, color-coded buttons and text.

# How to run:
1. Install pygame: pip install pygame
2. Run the game: python main.py
3. Click one of the Rock, Paper, or Scissors buttons

# How the AI works:
- The AI remembers what you pick
- It finds what you pick most often
- It picks the thing that beats your favorite choice
- The more you play, the smarter it gets!

## What you need:
- Python
- pygame library
